## 功能一：六爻占卜

### 输入格式

发送 6 组由 2 和 3 组成的数字（每组 3 个数字），或包含关键词：算卦、占卜、周易、易经、起卦、六爻

请心中默念所问之事，投掷硬币 6 次，每次 3 枚。记录每次的结果：
- **3** = 阳（字/正面）
- **2** = 阴（花/反面）

按顺序发送 6 组数字（从初爻到上爻）：

```
223 333 222 323 332 233
```

也可以在数字后面加上你的问题：

```
223 333 222 323 332 233 问事业发展
```

### 调用方法

```javascript
// 使用 calculateFiveGua() 计算六卦
const result = calculateFiveGua("223 333 222 323 332 233");
```

返回结果包含：
- `ben` — 本卦
- `bian` — 变卦
- `hu` — 互卦
- `zong` — 综卦
- `jiao` — 交卦
- `cuo` — 错卦
- `movingLines` — 动爻信息

## 功能二：八字排盘

### 触发方式

发送出生日期时间，包含关键词：八字、排盘、命盘、生辰、分析八字、八字详解

### 输入格式

- **公历生日**：`1990年5月15日20时出生`
- **农历生日**：`农历1999年12月18日20时`
- **简化格式**：`1999051520`

### ⚠️ 强制：必须调用的方法

**不得手算干支，必须使用以下方法获取精确数据。**

#### 步骤一：获取八字排盘

```javascript
// 农历生日用 getBaziFromLunar
const bazi = getBaziFromLunar(农历年, 农历月, 农历日, 时辰数字);

// 公历生日用 getBazi
const bazi = getBazi(公历年, 公历月, 公历日, 时辰数字);
```

示例：
```javascript
// 农历1999年12月18日20时
const bazi = getBaziFromLunar(1999, 12, 18, 20);

// 公历1990年5月15日20时
const bazi = getBazi(1990, 5, 15, 20);
```

`getBaziFromLunar` 返回值包含：
- `year` — 年柱干支
- `month` — 月柱干支
- `day` — 日柱干支
- `hour` — 时柱干支
- `bazi` — 完整八字字符串（空格分隔）
- `shichen` — 时辰名（如"戌时"）
- `lunarYear / lunarMonth / lunarDay` — 农历年月日
- `solarDate` — 公历转换日期
- `yearGan / yearZhi / monthGan / monthZhi / dayGan / dayZhi / hourGan / hourZhi` — 干支拆分

#### 步骤二：完整五行分析

```javascript
const analysis = analyzeBazi(bazi.year, bazi.month, bazi.day, bazi.hour);
```

`analyzeBazi` 返回值（`BaziAnalysis` 接口）包含：
- `bazi` — 完整八字字符串
- `yearZhu / monthZhu / dayZhu / hourZhu` — 四柱
- `riZhu` — 日主天干
- `riWuXing` — 日主五行
- `riShengXiao` — 生肖
- `geJu` — 格局（如"调候成格"）
- `wangShuai` — 旺衰信息
- `wuXingCount` — 五行统计
- `wuXingBalance` — 五行平衡描述
- `yongShen` — 用神
- `xiShen` — 喜神数组
- `jiShen` — 忌神数组
- `suggestions` — 结构化建议（方位、颜色、行业、日常习惯）

#### 步骤三：格式化输出

```javascript
const report = formatBaziAnalysis(analysis);
```

输出格式：


### 完整调用示例


## 功能三：每日黄历

### 触发方式

直接说"今日黄历"或"黄历"，或包含关键词：每日宜忌、今日吉凶

### ⚠️ 强制：必须调用的方法

```javascript
const huangLi = getHuangLi(公历年, 公历月, 公历日);
const report = formatHuangLi(huangLi);
```

示例：
```javascript
// 查询今日黄历
const today = new Date();
const huangLi = getHuangLi(today.getFullYear(), today.getMonth() + 1, today.getDate());
const report = formatHuangLi(huangLi);
```

`getHuangLi` 返回值包含：公历日期、农历日期、干支、生肖、纳音五行、彭祖百忌、冲煞、吉神方位、今日宜忌、九星、星宿等完整信息。

### 八字深度推理步骤

#### 第一步：十神推演

从 `analyzeBazi()` 返回的四柱干支出发，逐一推导每个天干与日主（`riZhu`）之间的十神关系。

**十神关系速查表**（以日主为中心）：

| 天干 | 生日主 | 被日主生 | 克日主 | 被日主克 | 同类 |
|------|--------|----------|--------|----------|------|
| 木 | 印星 | 食伤 | 财星 | 官杀 | 比劫 |
| 火 | 印星 | 食伤 | 财星 | 官杀 | 比劫 |

| 水 | 印星 | 食伤 | 财星 | 官杀 | 比劫 |

十神速记：同性相生为枭印，异性相生为正印；同性相克为七杀，异性相克为正官。

**十神性格含义**：
- **比肩/劫财**：自信、独立、竞争、固执
- **食神/伤官**：才华、表达、创新、叛逆
- **正财/偏财**：理财观、物质欲望、勤俭或大方
- **正官/七杀**：责任心、压力、规矩、野心
- **正印/枭神**：学习、依赖、思考、懒散


### 六爻深度推理步骤

#### 第一步：世应关系分析

`calculateFiveGua()` 返回的本卦中，世爻代表求测者，应爻代表对方或环境。

| 世应关系 | 含义 |
|----------|------|
| 世应相生 | 双方关系和谐，进展顺利 |
| 世应相克 | 双方存在矛盾、阻碍或冲突 |
| 世克应 | 我方主动，对方配合度低 |
| 应克世 | 对方主动施压，我方被动 |
| 世应相冲 | 关系变动大、分歧严重或即将分离 |
| 世应相合 | 双方关系稳固，有合作基础 |

#### 第二步：动变深度解读

动爻是事情变化的关键点，需要逐爻解读：

1. 定位动爻位置（初爻至六爻）
2. 查该爻的爻辞（来自卦爻辞库或古法解读）
3. 判断动爻的五行属性对用神的影响

动爻解读框架：
- **动爻生世/用神**：事情朝有利方向发展，贵人相助
- **动爻克世/用神**：遇阻或小人，防是非口舌
- **动爻化回头克**：原本有利的变化转坏，需警惕
- **动爻化进神**：事情越来越旺，积极信号
- **动爻化退神**：事情越来越弱，需抓紧行动

#### 第三步：六亲归属

六亲断事速查（以世爻为核心）：

| 六亲 | 持世时的含义 |
|------|-------------|
| 父母爻持世 | 多与房产、证件、合同、父母相关；事多操心 |
| 官鬼爻持世 | 多与事业、职位、丈夫/男性压力相关；责任心重 |
| 兄弟爻持世 | 多与合伙、竞争、花费、口舌相关；防破财 |
| 妻财爻持世 | 多与钱财、妻子、物质相关；财运为重 |
| 子孙爻持世 | 多与子女、喜悦、创作、贵人相关；诸事顺遂 |

#### 第四步：卦象综合判断

1. **先看卦名本身**：卦名已经传达了核心含义
   - 既济（已经成功）→ 事情已有眉目，但需防后患
   - 未济（尚未成功）→ 事情还在发展中，需继续努力
   - 师（众）→ 多人合作或有纠纷
   - 同人（会同）→ 合作、结盟
   - 大有（有很多）→ 财运好、收获大

2. **再看本卦与变卦的关系**：
   - 本卦生变卦：现状推动事情往某方向发展
   - 本卦克变卦：现状制约或阻碍目标
   - 变卦生本卦：外部因素有利于现状
   - 变卦克本卦：外部因素对现状不利

3. **综合卦辞**：将卦象核心含义与用户所问之事结合，生成有针对性的解读

#### 第五步：针对问题的明确解答

六爻解读不能只泛泛说"事业顺利"或"感情波折"，必须结合用户实际问题给出具体判断：

- **问事业**：目前进展如何？阻碍在哪里？何时转机？应该进还是退？
- **问感情**：对方态度如何？感情现状？有无第三竞争？进展时机？
- **问财运**：求财时机？方向？风险？忌讳？
- **问健康**：有无大碍？何时康复？注意事项？
- **问出行**：宜不宜出发？吉凶如何？注意事项？

**回答模板**：
```
【针对您的问题】
现状（本卦）：...（描述当前状态）
关键动爻：...（解释最重要的变化信号）
阻碍/助力：...（具体分析有利和不利因素）
趋势判断：...（事情往好还是坏的方向发展）
建议：...（明确告诉用户应该怎么做、什么时候行动）
时机提示：...（如果卦象有提示，给出时间范围）
```
