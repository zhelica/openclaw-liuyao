## 触发方式

发送 6 组由 2 和 3 组成的数字（每组 3 个数字），或包含关键词：算卦、占卜、周易、易经、起卦、六爻

## 输入格式

请心中默念所问之事，投掷硬币 6 次，每次 3 枚。记录每次的结果：
- **3** = 阳(字/正面)
- **2** = 阴(花/反面)

按顺序发送 6 组数字（从初爻到上爻）：

```
223 333 222 323 332 233
```

也可以在数字后面加上你的问题：

```
223 333 222 323 332 233 问事业发展
```

## 六卦含义

- **本卦**：当前状态，代表现状
- **变卦**：未来变化，代表事情发展的结果
- **互卦**：变化过程，代表事情发展中的中间状态
- **综卦**：换位思考，从不同角度看待问题
- **交卦**：内外调换，关注内外部环境的对调变化
- **错卦**：潜在对立面，提醒注意的风险和挑战
