# 六爻易经 (Liuyao)

基于《周易》六爻占卜法的 OpenClaw Skill，提供本卦、变卦、互卦、综卦、错卦五维深度解读。

## 功能特性

- **六爻起卦**：支持 6 组数字输入（每组 3 个数字，2=阴/花，3=阳/字）
- **五卦计算**：
  - 本卦：当前状态
  - 变卦：未来变化
  - 互卦：变化过程
  - 综卦：换位思考
  - 错卦：潜在对立面
- **智能解卦**：包含卦象含义和综合建议
- **动爻识别**：自动识别老阴(6)、老阳(9)动爻

## 安装

将 Liuyao目录复制到你的 OpenClaw 插件的 skills 目录下：

```json
{
    "skills": {
    "entries": {
      "liuyao": {
        "enabled": true,
        "config": {
          "name": "六爻易经",
          "description": "基于《周易》六爻占卜法的智能解卦技能",
          "path": "/home/admin/.openclaw/workspace/skills/LiuYao"
        }
      }
    }
  }
}
```

## 使用方法

### 输入格式

请心中默念所问之事，投掷硬币 6 次，每次 3 枚。记录每次的结果：
- **3** = 阳(字/正面)
- **2** = 阴(花/反面)

按顺序发送 6 组数字（从初爻到上爻）：

```
223 333 222 323 332 233
```

你也可以在数字后面加上你的问题：

```
223 333 222 323 332 233 问事业发展
```

## 算法说明

1. **求和转换**：每组 3 个数字相加，得到 6, 7, 8, 9
   - 6 = 老阴（动，变阳）
   - 7 = 少阳（静爻）
   - 8 = 少阴（静爻）
   - 9 = 老阳（动，变阴）

2. **二进制转换**：7,9 → 阳(1)，6,8 → 阴(0)

3. **五卦计算**：
   - 本卦：直接映射原始数据
   - 变卦：6→7(1)，9→8(0)，7/8 不变
   - 互卦：取 2,3,4,3,4,5 爻重组
   - 综卦：上下反转
   - 错卦：阴阳取反
